Below you'll find empty'ish html files with different ad-yield-managing companies' js loaded. It's built, so that I can test their impact on First Input Delay (FID). They are loaded from the same server, so that uBlock doesn't block them (but block pretty much everything else that they import). This way I can isolate their impact on FID. Instructions:
  1. use Google Chrome
  2. have some adblock installed (I'm using uBlock) and activated in the incognito mode. The goal is to isolate the impact of your JS lib, so we want all the other ad-related libraries blocked.
  3. load this page in the incognito mode
  4. click on one of the links below, to load an empty page with that imports the ad js library from a given company
  5. open dev tools, go to the audits tab.
  6. run the audit. We're looking at (and only at) the Max Potential First Input Delay (it's in the performance report)
Please note that I may not be importing everything that's needed in some cases, I simply looked at what's imported by those companies' clients' websites. I may havge made mistakes. Pages:
